В данном репозитории ведётся восстановление и разработка звуков для проекта FMOD игры Prime World. Результатом кропотливой работы будет проект, готовый для экспорта в игру через FMOD Designer. 

#Лицензия
Данный репозиторий является частью репозитория Prime World от Nival https://github.com/nival/Prime-World. Вы можете использовать наработки из данного репозитория согласно лицензии, составленной Nival для игры Prime World. Использование наработок потребует от Вас публикации в публичный доступ контента и исходного кода в 2-х недельный срок. Согласно внутренним договорённостям, разрешается использование наработок без требования к открытости, но по истечении 1 года после внедрения наработок в сервер Prime World Classic. Просьба упомянуть авторство команды Prime World Classic в первом после внедрения наработок публичном посте и всех его дубликатах. 

Коммерческое использование данного проекта ограничено основной лицензией на игру Prime World.
